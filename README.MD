### Installation
1. Download latest release archive from releases.
2. Extract it somewhere. The software is portable (as long as it was launched once in a new directory if was moved).
3. Launch **WebRichPresence*W*.exe**.

### Usage
0. First of all, the software is not required to be manually launched. It is launched by a browser extension when it is needed.
1. Install compatible browser extension.
2. Open a page that the extension can work with.
3. Accept browser's webrichpresence:// launching prompt (tick the checkbox to make it work seamless).
